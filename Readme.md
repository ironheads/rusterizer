No dependencies were used, writtent entirely in Rust from scratch.

## Build
### Locally
#### Prerequisites
 - rustup
 - cargo

```bash
> rustup update
> rustup default nightly
> cargo run --features=local
```

### Web
#### Prerequisites
 - trunk
 - python3
 - rustup
 - cargo

```bash
> rustup update
> rustup default nightly
> rustup target add wasm32-unknown-unknown
> ./build.sh
```
